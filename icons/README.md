# Icons Directory

This directory should contain the extension icons in the following sizes:

- `icon16.png` - 16x16 pixels (toolbar icon)
- `icon48.png` - 48x48 pixels (extension management page)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

## Icon Requirements

- **Format**: PNG with transparency
- **Style**: Simple, recognizable design
- **Colors**: Should work well with the extension's purple gradient theme
- **Design**: Focus-related imagery (target, timer, brain, etc.)

## Placeholder Icons

For development, you can create simple placeholder icons or use online icon generators. The extension will work without these icons, but they improve the user experience.

## Creating Icons

You can create icons using:
- Online tools like Canva, Figma, or Icon8
- Design software like Adobe Illustrator or Sketch
- Icon generators like Favicon.io or RealFaviconGenerator
