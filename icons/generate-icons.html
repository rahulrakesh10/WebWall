WebWall Icon Generator

Generate the required icon files for the extension

Instructions:

  1. Click "Generate Icons" to create the icon previews
  2. Right-click each canvas and select "Save image as..."
  3. Save as:
    • icon16.png (16x16 pixels)
    • icon48.png (48x48 pixels)
    • icon128.png (128x128 pixels)
  4. Place all three files in the icons/ directory
